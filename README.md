<!-- GETTING STARTED -->
## Getting Started



### Prerequisites

* Install Dependencies 
  ```sh
  pip install -r requirements.txt
  ```

### Installation


2. Clone the repo
   ```sh
   git clone https://github.com/ashen8810/Sinhala-hate-speech.git
   ```

2. Run Flask app
   ```sh
   cd Sinhala-hate-speech
   python flaskapp.py
   ```
   or
   
    ```sh
   cd Sinhala-hate-speech
   streamlit run app.py
   ```
